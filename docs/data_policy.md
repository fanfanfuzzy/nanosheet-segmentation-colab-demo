# Data Policy / データ公開方針

## English

This public repository does **not** contain:

- Real experimental TEM images
- Manual annotations derived from real experiments
- Unpublished experimental datasets
- Real experimental metadata (file names, instrument conditions, sample conditions)
- Model weights trained on private/real data
- Prediction results overlaid on real experimental images

All images, masks, predictions, and sample metrics included in this repository are:

- Synthetically generated for educational purposes, OR
- Derived entirely from synthetic data

This policy exists because public repositories, once shared, can be cloned, forked, and cached indefinitely. Real experimental data should never be committed to a public repository.

## 日本語

このリポジトリには以下のデータは含まれて**いません**：

- 実際の実験TEM画像
- 実験データに基づく手動アノテーション
- 未公開の実験データセット
- 実験メタデータ（ファイル名、装置条件、試料条件）
- 実験データで学習したモデルの重み
- 実験画像に重ねた予測結果

このリポジトリに含まれるすべての画像・マスク・予測結果・指標は：

- 教材用に人工的に生成されたもの、または
- 合成データのみから生成されたもの

です。

公開リポジトリは一度公開するとclone・fork・cacheされるため、実験データは絶対に含めないという方針です。
